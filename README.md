# Mayros Corretora de Seguros

Landing page institucional da Mayros Corretora de Seguros — Porto Alegre/RS.

## Sobre

Single-page application desenvolvida em HTML + React 18 + Tailwind CSS, com design system liquid-glass inspirado em interfaces premium. Sete seções: Hero, Sobre, Serviços, Depoimentos, FAQ, Contato e Footer.

## Stack

- HTML5 estático
- React 18.3.1 (via CDN UMD)
- @babel/standalone 7.29.0 (JSX in-browser)
- Tailwind CSS (CDN com config inline)
- Fontes: Fraunces (heading) + Inter (body)
- Ícones: SVG inline (lucide-style)
